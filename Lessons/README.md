# Lessons in Software Engineering

A centralized repository for high-level architectural insights, engineering principles, and lessons learned through AI-orchestrated development.

## Directories

- [Architectural Thinking](./Architectural-Thinking/) - Systems design, infrastructure, and core engineering principles.
- [AI Orchestration](./AI-Orchestration/) - Strategies for managing LLMs, prompt engineering, and "Agentic" development.
- [Project Post-Mortems](../) - Link back to project-specific logs for empirical data.

## Core Philosophies
1. **The Scientific Method:** Formulate hypotheses before acting.
2. **Reverse Engineering Era:** Learn by challenging AI-generated systems.
3. **Complexity over Syntax:** Measure growth by systems managed, not lines typed.
